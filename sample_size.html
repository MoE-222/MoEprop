Sample Size Determination

Purpose of this calculator

Determine required sample size (n) to achieve specified confidence level with predefined margin of error.

General Formula:

n = (Zα/2 × σ / e)²
e = Allowable margin of error
σ = Standard deviation (estimated)

Application Example

Mean and standard deviation of quality points were 2.6 and 0.3 respectively. What sample size is required for 95% confidence with error less than 0.05?

n = (1.96 × 0.3 / 0.05)² = (11.76)² = 138.3 ≈ 139

Tips:

  • If σ unknown, use conservative estimate or value from similar studies
  • For proportions, use P = 0.5 for maximum variance (largest sample size)
  • Always round up the result to ensure sufficient sample size
