Confidence Interval for Proportion

Population Proportion | P̂

Conditions for using this formula

  • We use normal approximation
  • Must have nP̂ ≥ 5 and n(1-P̂) ≥ 5
  • Large sample size
  • Binary data (success/failure)

General Formula:

P̂ ± Z × √[P̂(1-P̂)/n]
where P̂ = X/n
X = number of successes

Application Example

In a random sample of 500 families in Hamilton, Canada, 340 families owned color TVs. The 95% confidence interval for the true proportion:

P̂ = 340/500 = 0.68
0.68 ± 1.96 × √(0.68×0.32/500) = 0.68 ± 0.041
0.639 < P < 0.721 (i.e. 63.9% to 72.1%)
